Bus Reservation System

Project Description

The Bus Reservation System is a Java-based console application developed to manage bus ticket bookings. The system allows users to view available seats, book tickets, cancel reservations, and check reservation status.

Features

- User Login Authentication
- View Available Seats
- Book Bus Tickets
- Cancel Booked Tickets
- Check Reservation Status
- Simple Console-Based Interface

Technologies Used

- Java
- NetBeans IDE
- Scanner Class for User Input

How to Run

1. Open NetBeans IDE.
2. Create a Java Application Project.
3. Copy the source code into Main.java.
4. Save the file.
5. Click Run or press F6.
6. Enter the login credentials.
7. Use the menu options to manage reservations.

Menu Options

1. View Seats
2. Book Ticket
3. Cancel Ticket
4. Reservation Status
5. Exit
